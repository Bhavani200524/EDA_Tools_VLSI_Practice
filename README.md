# EDA_Tools_VLSI_Practice
This Certificate contains work completed during my Winter Internship on VLSI Design at SR University, Warangal. It includes RTL design, digital IC design flow, simulation results, and exposure to industry-standard EDA tools.

##  Overview
This repository contains my hands-on practice and learning using Electronic Design Automation (EDA) tools for VLSI design.  
It focuses on understanding the complete digital design flow from RTL to simulation and analysis.

##  Objectives
- Learn and practice VLSI design using EDA tools  
- Understand the RTL to GDSII design flow  
- Perform simulation and verification of digital circuits  
- Gain practical exposure to industry-standard tools  

##  Tools Used
- ModelSim  – Simulation  
- Xilinx Vivado  – Design & Synthesis  
- Synopsys Tools – VLSI Design Flow  
- GTKWave – Waveform Analysis  


##  Work Done
- Simulated digital circuits using Verilog  
- Performed functional verification  
- Analyzed waveforms and outputs  
- Practiced RTL design and synthesis flow  


##  VLSI Design Flow Covered
1. RTL Design (Verilog)
2. Simulation & Functional Verification
3. Synthesis
4. Layout (Basic Understanding)
5. Testing & Analysis


## Skills Gained
- RTL Design  
- Verilog Coding  
- Simulation & Debugging  
- EDA Tool Usage  
- Digital System Design  


##  Outcome
Gained practical knowledge of VLSI design using EDA tools and understood the real-world chip design workflow.
